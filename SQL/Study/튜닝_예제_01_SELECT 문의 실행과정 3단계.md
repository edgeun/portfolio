### 튜닝 1.  SELECT 문의 실행과정 3단계

- select 문의 처리과정

```sql
select ename, sal
 from emp
 where ename = 'SCOTT';
```

1. 파싱(parsing) : SQL을 기계어로 변환하는 작업
2. 실행(execute) : 실행계획대로 검색하는 데이터를 DB에서 찾는 과성
3. 패치(fetch) : DB에서 찾은 결과를 SQL을 수행한 유저 프로세서에게 전달
